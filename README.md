# wechat-cli
微信 命令行版（简陋）

- 只支持 python3
- 支持 Windows Cmd / Linux / cygwin / Mac

## 功能：

- 和别人聊天
- 接收消息（文字）
- 按照名字和微信号精确搜索好友

## 依赖

```
pip3 install itchat
```
